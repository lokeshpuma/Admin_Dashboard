# GitHub Pages Deployment Configuration

This Next.js project is configured for GitHub Pages static deployment.

## Key Configuration

- **Output Mode**: Static export (`output: 'export'`)
- **Base Path**: `/ForesTracker-2` (repository name)
- **Image Optimization**: Disabled for static hosting

## Build Process

Run `pnpm build` to generate the static site in the `out/` directory.

## Deployment Steps

1. Build: `pnpm build`
2. Commit changes: `git add . && git commit -m "Deploy"`
3. Push to GitHub: `git push origin main`
4. Enable GitHub Pages in repository settings

The deployment will be available at: `https://lokeshpuma.github.io/ForesTracker-2/`
